## Here is some Example of Code block using prims syntax heighlighting

Here's a basic Markdown syntax guide along with some extended syntax elements:

1. **Basic Elements**:
   - **Heading**: `# H1`, `## H2`, `### H3`
   - **Bold**: `**bold text**`
   - **Italic**: `*italicized text*`
   - **Blockquote**: `> blockquote`
   - **Ordered List**: `1. First item`, `2. Second item`, `3. Third item`
   - **Unordered List**: `- First item`, `- Second item`, `- Third item`
   - **Code**: \`code\`
   - **Horizontal Rule**: `---`
   - **Link**: `[title](https://www.example.com)`
   - **Image**: `![alt text](image.jpg)`

2. **Extended Syntax**:
   - **Table**:
     ```markdown
     | Syntax | Description |
     | ----------- | ----------- |
     | Header | Title |
     | Paragraph | Text |
     ```

<CodeBlock code={
`
// (home)/components/CodeBlock.tsx
import React from 'react';
import Prism from 'prismjs';
import 'prismjs/components/prism-jsx';
import 'prismjs/themes/prism-okaidia.css';
import DOMPurify from 'isomorphic-dompurify';
import { BackgroundGradient } from '@/components/ui/background-gradient';

interface CodeBlockProps {
  children: string;
}

const CodeBlock: React.FC<CodeBlockProps> = ({ children }) => {
  const language = 'jsx';
  const highlightedCode = Prism.highlight(children, Prism.languages[language], language);

  return (
    <BackgroundGradient className="rounded-[22px] p-4 sm:p-10 bg-white dark:bg-zinc-900">
      <pre >
        <code
          }
          dangerouslySetInnerHTML={{
            __html: DOMPurify.sanitize(highlightedCode),
          }}
        ></code>
      </pre>
    </BackgroundGradient>
  );
};

export default CodeBlock;
`
} language="jsx" />
